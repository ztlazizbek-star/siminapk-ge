# User API - Инструкция по установке

## Шаг 1: Создание таблицы пользователей

1. Откройте **phpMyAdmin** в панели управления хостингом
2. Выберите базу данных `cg47924_simin`
3. Перейдите на вкладку **SQL**
4. Выполните SQL скрипт из файла `scripts/create-users-table.sql`

## Шаг 2: Загрузка API файла

1. Загрузите файл `user.php` в папку `/simin/` на вашем сервере
2. Полный путь должен быть: `tajstore.ru/simin/user.php`

## Шаг 3: Проверка работы

Откройте в браузере или через Postman:

**Регистрация пользователя:**
\`\`\`
POST https://tajstore.ru/simin/user.php
Content-Type: application/json

{
  "action": "register",
  "name": "Иван",
  "phone": "+992123456789",
  "address": "ул. Рудаки, 123"
}
\`\`\`

**Проверка пользователя:**
\`\`\`
POST https://tajstore.ru/simin/user.php
Content-Type: application/json

{
  "action": "verify",
  "phone": "+992123456789"
}
\`\`\`

## Готово!

Теперь приложение будет:

- Сохранять новых пользователей в БД при регистрации
